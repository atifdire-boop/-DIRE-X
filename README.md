# 
## 📋 Overview

**DIRE-X** is a comprehensive, professional-grade cybersecurity toolkit containing **20 advanced tools** for ethical hacking, penetration testing, and security research. Built specifically for Termux but compatible with Linux, macOS, and Windows.

## 🚀 Features

- ✅ 20 Professional Security Tools
- ✅ Modular Architecture
- ✅ Termux Optimized
- ✅ Colorful Terminal Interface
- ✅ Auto-Update System
- ✅ Multi-threaded Operations
- ✅ Custom Wordlists
- ✅ Report Generation

## 📦 Installation

```bash
# Clone repository
git clone https://github.com/YOUR_USERNAME/DIRE-X.git

# Navigate to directory
cd DIRE-X

# Run installer
python3 install.py

# Launch DIRE-X
python3 dire-x.py
 DIRE-X
